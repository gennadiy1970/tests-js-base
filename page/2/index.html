/*
  Инкремент (++i), декремент(--i) -
  эти сложные слова описывают всего лишь операцию
  увеличения и уменьшения значения переменной на единицу.
  А постфиксная(i++) и префиксная(--i) форма описывают момент
  когда именно это изменение произойдет:
  после (i++) или до (++i) текущей операции.
  // var i = 0, j = 0; ++i + 2 === 3;  j++ + 2 === 2;
  // var l = 0, m = 0; --l + 2 === 1;  m-- + 2 === 2;
